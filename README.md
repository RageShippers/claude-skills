# Claude Skills

This repository contains the skills that are used by Claude to interact with the user.

## Skills

- [/linear-build](linear-build.md) - Start active work on Linear task and track progress
- [/linear-pr](linear-pr.md) - Create a PR for a Linear task
- [/linear-status](linear-status.md) - Check the status of a Linear task

## Usage

Copy the contents of the skill file into the `~/.claude/commands` folder in your home Claude directory.
